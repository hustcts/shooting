# 🎯 Shooting Game - 射击游戏

一个使用 HTML5 Canvas 和 JavaScript 开发的开源射击小游戏。

![Shooting Game](screenshot.png)

## 🎮 在线试玩

[点击这里开始游戏](https://hustcts.github.io/shooting/)

## ✨ 特性

- 🎨 精美的渐变背景和粒子效果
- 🎯 流畅的射击体验
- 📈 难度递增的关卡系统
- 💥 炫酷的爆炸特效
- 📱 支持键盘和鼠标/触摸控制
- ❤️ 生命值系统

## 🕹️ 游戏操作

| 操作 | 键盘 | 鼠标/触摸 |
|------|------|----------|
| 向左移动 | ← 或 A | - |
| 向右移动 | → 或 D | - |
| 射击 | 空格键 | 点击屏幕 |

## 🎯 游戏玩法

1. **开始游戏**：点击"开始游戏"按钮
2. **移动飞机**：使用 ← → 键或 A D 键控制左右移动
3. **射击敌人**：按空格键或点击屏幕发射子弹
4. **消灭敌人**：击中敌人获得分数，防止敌人撞击你或飞出屏幕
5. **升级**：分数达到 100 的倍数时升级，敌人速度和生成频率会增加
6. **生存**：生命值归零时游戏结束

## 💡 技术实现

- **HTML5 Canvas**：游戏渲染
- **JavaScript ES6**：游戏逻辑
- **CSS3**：UI 样式和动画
- **无依赖**：纯原生实现，无需任何库
